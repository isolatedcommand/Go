How a Go link works

1

Visit a short link

Someone opens go.isolatedcommand.com/github.

2

The Worker looks it up

A Cloudflare Worker reads the path and finds the matching destination in Workers KV.

3

Instant redirect

The Worker returns a 301 redirect to the real URL — served from the edge, worldwide.
