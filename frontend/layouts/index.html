How a Go link works

1

Create or visit a link

Shorten any URL above, or open an existing go.isolatedcommand.com/github.

2

Validated & screened

A Cloudflare Worker checks the URL for unsafe schemes, private-network SSRF and blocked content before storing it in Workers KV.

3

Instant, counted redirect

Matching links redirect from the edge worldwide — and every click is counted.

Built in the open by DevComm

Go is a community project of the Isolated Command Developer Community (DevComm). The code, security model and configuration are all open — fork it, self-host it, or contribute a fix. No servers, no database to run: just Cloudflare Workers and Workers KV.
