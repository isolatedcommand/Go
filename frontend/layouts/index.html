Everything a shortener should be

Edge-fast redirects

Links resolve from Cloudflare's global network, close to every visitor — no origin server, no cold starts.

Secure by default

Every destination is validated: unsafe schemes, private-network SSRF, redirect loops and blocked content are rejected.

Custom links

Pick a memorable name, or get a random code. Reserved words are protected so nothing collides.

Built-in analytics

Track clicks, creation date and last-accessed time for every link, updated live.

Abuse-protected

Per-IP rate limiting with hashed identifiers keeps the service clean — and admin is gated by Cloudflare Access.

Open source

Read the code, file an issue, or self-host your own instance. Built in the open by DevComm.

How a Go link works

1

Create or visit a link

Shorten any URL above, or open an existing go.isolatedcommand.com/github.

2

Validated & screened

A Cloudflare Worker checks the URL for unsafe schemes, SSRF and blocked content before storing it in Workers KV.

3

Counted redirect

A friendly 5-second interstitial shows the destination, then sends the visitor on — and counts the click.
